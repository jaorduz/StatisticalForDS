Mode. It is the prominent peak in the distribution (sometimes it is identified as the highest rectangle) in the histogram. Besides, it is the value with the most occurrences in the data set. But this definition has some issues, such as, it is uncommon to have repetitions in the data set. On the other hand, the data set can have more peaks, e.g. one mode (unimodal), two mode (bimodal), or several modes (multimodal).
