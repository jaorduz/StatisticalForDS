  • Bar plot. It is the common plot to show a categorical variable.
  • Row proportions. It is a way to show the quotient between one case compare to the total. This can be done horizontal.
  • Column proportions. It is a way to show the quotient between one case compare to the total. This should be done vertical.
  • bar plot with two variables is an extension for the bar plot. It is also called stacked bar plot. This kind of plot represents different groups of one varible stacked.
  • Mosaic plots. This kind of plot uses areas to stand for the number of cases in each category.
  • Pie chart. It is another to represent the information, similarly to bar plot, but in a circle shape.
