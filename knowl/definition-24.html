Definition 2.1.0.7.

  1. Nominal scale level These kind of data can be measured using categories, such as, names, labels, or yes or not answers.
  2. Ordinal scale level These kind of data can be ordered such as a list of top five of Zoos in the USA. There is not a scale to measure the difference between two Zoos. Another example, responses such as sad, neutral, and happy. There is an order, but there is not an scale at all.
  3. Interval scale level. It has a definite ordering, but we cannot compare data. Examples: Different scales to measure temperature.
  4. Ratio scale level. It can be order-definited, and has a zero point and it represents same information for different data. Example: points to grade exams.
